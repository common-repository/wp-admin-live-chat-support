Provides live technical support and troubleshooting via a chat module from the WP Admin interface.

== Description ==

Live chat plugin that displays on the Admin side of a WordPress website and provides a connection to Burlington Byte’s technical support service. Chat box is positioned conveniently in the bottom right-hand corner and is available for activation from any page within the Admin panel of the website.

On click, a pop-up box opens and user can submit requests, chat with a live support person, and receive technical assistance while continuing to work within the WordPress admin side of the website.

Support services are provided for Burlington Bytes clients. If you aren’t a current Burlington Bytes client, you can still install the plugin. After install, you’ll be redirected to one of our support specialists for assistance. For information on our support services, please visit [our website](http://www.burlingtonbytes.com/wp-admin-live-chat-support)

Please note: These services are only enabled in supported websites. For full documentation and details, please refer to the Burlington Bytes website.

== Installation ==

Please refer to full installation instructions at [our website](http://www.burlingtonbytes.com/wp-admin-live-chat-support)

== Frequently Asked Questions ==

If you have any issues installing this plugin, please contact us at support@burlingtonbytes.com. If you already have purchased support from us, please provide your support activation code in the message body of the email.

You can also visit [the FAQ page on our website](http://www.burlingtonbytes.com/wp-admin-live-chat-support/faq/) for more information.
